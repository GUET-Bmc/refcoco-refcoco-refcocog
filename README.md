# Referring Expression Datasets

This repository contains resources and/or code for handling referring expression datasets.

## Datasets

The following datasets are supported in this project:

- **RefCOCO** 
- **RefCOCO+**
- **RefCOCOg**

These datasets were originally collected by UNC and are widely used for referring expression comprehension tasks.
